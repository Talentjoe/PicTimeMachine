# PicTimeMachine · 图片时光机

## Overview · 项目简介
PicTimeMachine is a pure front-end React project that visualizes geo-tagged photos on an interactive map, letting you revisit memories in the places where they were captured.
PicTimeMachine 是一个纯前端的 React 项目，能够在互动地图上展示带有地理信息的照片，让你在地图上重温拍摄时的足迹。

## Features · 功能亮点
- Interactive map browsing of your photo collection / 通过互动地图浏览个人照片集
- Automatic extraction of EXIF geo-information for positioning / 自动提取 EXIF 地理信息以定位照片
- Time-based filtering to revisit trips chronologically / 按时间筛选，按旅程顺序回顾回忆
- Built with React and modern mapping libraries for smooth performance / 基于 React 与现代地图库构建，带来流畅体验

## Getting Started · 快速开始

### Prerequisites · 前置条件
- Node.js 16 or newer
- npm (bundled with Node.js)

需要准备 Node.js 16 或更高版本，并使用自带的 npm 包管理器。

### Installation · 安装依赖
```bash
npm install
```
运行上述命令安装项目依赖。

### Development Server · 启动开发服务器
```bash
npm start
```
在本地启动开发服务器，默认地址为 http://localhost:3000。

### Production Build · 构建生产版本
```bash
npm run build
```
生成可部署的静态文件。

### Tests · 运行测试
```bash
npm test
```
执行内置的测试套件。
